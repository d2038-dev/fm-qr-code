### What I learned

Learned to set body to full viewport height:

```css
body {
  min-height: 100vh;
}
```

and using flexbox to expand wrapper:

```css
body {
  display: flex;
  flex-direction: column;
}

.wrapper {
  flex: 1;
}
```
